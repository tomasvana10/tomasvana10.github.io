Keybinds & Accessibility

Several keybinds have been implemented for ease of access.

  • Submit Guess: ENTER
  • Place Peg: 1, 2, ...
  • Delete Peg: BACKSPACE
  • New Game: SHIFT + ENTER
  • Manual Peg Input Mode: SHIFT + CLICK (on code peg of choice)

To further improve accessibility, all clickable elements are focusable with the TAB key. The functions of these elements can be invoked by pressing ENTER.

How To Play

Game Rules:
  • The user plays as the codebreaker, and must determine a code that is randomly assembled by the codemaker â€” an algorithm.
  • NOTE: Once this page is loaded, a random code is generated, and the game begins. You do not need to press the "New Game" button initially.
  • A player has a fixed amount of guesses of the code before they fail. The amount of guesses defaults to 10.
  • A player's guess consists of coloured code pegs, which much occupy all the available guess slots. The amount of guess slots defaults to 4 (and thus, the codemaker's code also consists of 4 slots).
  • Once the player makes their guess, the algorithm will respond by placing key pegs, which are placed by comparing the player's guess with the codemaker's code using the following guidelines:
    • One black key peg for each code peg that is the correct colour and in the correct position.
    • One grey key peg for each code peg that is the correct colour but in the wrong position.
    • NOTE: In an example scenario where the codemaker's code contains one red peg, and the user's code contains two red pegs, only one black key peg is placed.
    • NOTE: Key pegs are not placed in any particular order.
To Play:
  1. Click on the code pegs under "Code Peg Picker" to arrange your code. You can use the "Delete" button to remove code pegs if you wish.
  2. Press the "Guess" button to submit your code. The algorithm will provide you with feedback using key pegs.
  3. To start a new game at any time, press the "New Game" button.
